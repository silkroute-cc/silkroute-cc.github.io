Touch'n Go eWallet

马来西亚支付受国家政策、银行接口的开放权限及互联网发展等因素影响，移动支付发展没有这么快，目前当地人使用比较多的手机支付主要有Touch'n Go eWallet、Boost Wallet、Maybank Pay（手机银行）。

今天主要给大家分享马来西亚当地“支付宝”——Touch'n Go eWallet！

TNG发展历程 TNG预付卡 TNG是马来西亚最大的预付卡公司，拥有2000万TNG实体卡用户。TNG卡绑定着马来西亚人的个人ID，TNG母公司的一大股东是CIMB（马来西亚联昌国际银行），也是马来西亚第二大银行。

Touch'n Go，马来西亚一触即通卡，目前，每日有上百万的大马人使用一触即通卡，缴付过路费、公共交通费、停车费等。

2017年7月，蚂蚁金服与马来西亚联昌国际银行旗下的Touch'n Go（TNG）公司签署协议，组建合资公司推出本地钱包Touch'n Go eWallet。

Touch'n Go eWallet

TNG eWallet充值方式 使用您的CIMB银行、Maybank、HSBC、RHB、Bank Islam、Bank Simpanan nanational或任何使用FPX网上银行、柜台现金或信用卡/借记卡绑定您的电子钱包。

使用TNG电子钱包进行日常消费

预付充值 直接从您的TNG电子钱包充值U Mobile、Hotlink、Digi、YES、Xmax、Tune Talk、XOX或Altel预付费。

网上购物 用你的TNG电子钱包从Lazada、PrestoMall和淘宝网购买日常必需品或满足你的购物需求。满足你的食物渴望与送货和饥饿，支付与你的电子钱包获得免费的食物送货！

邮费和水电费账单 用你的电子钱包结算你的Maxis，U Mobile，Celcom, YES，redONE，Unifi，Astro账单！支付水电费，如水费，甚至PTPTN贷款偿还全部从电子钱包本身！

杂货店和药店 需要快速进行必要的跑步吗？从乐购、Mydin、Jaya杂货店、7-11、KK Mart、99 Speedmart、MyNews等处购买。从Caring、Watsons、Guardian、BIG Pharmacy、AA Pharmacy、Alpro等获得您的医疗保健需求。

一步汇款 只需输入一个电话号码，就可以立即将钱转给其他Touch'n Go eWallet用户。

使用PAYDIRECT/RFID支付通行费 直接从应用程序支付通行费！只需将您的Touch'n Go card添加到应用程序中，当您点击收费时，您的电子钱包余额将被扣除，而不是您的卡！

付停车费 无论你在哪里，都要为你的街道停车费买单！现在可在DBKL、MPSJ、普特拉贾亚管理局、Daerah Kuala Langat少校、M.P.Kota Bharu等处购买！

快预定电影 选择一个座位从主要的电影院，如GSC，TGV，MBO和更直接的电子钱包，并支付在两个步骤！

预订机票 计划您的航班，并在电子钱包内预定，也分两步！

使用TNG钱包，你可以在便利店，加油站和药店，即KK-Mart，Mydin Caring Pharmacy（药房）快速完成付款！

TNG还在不断更新电子钱包，以增加更多的功能和让在线支付变得更简单！

Touch'n Go eWallet促销活动

  • 在LAZADA商店购物，并使用TNG eWallet进行首次付款，以享受首次获得RM 6的现金回扣！
  • 在Paradigm Mall PJ购物时，消费满RM100即可获得RM30的现金券！
  • 无需最低消费即可在Flower Chimp享受18％的折扣！

Touch'n Go eWallet特点和优势

  • 记录每笔交易，密切关注您的费用；
  • 以退款保证为后盾，增强的安全功能使您放心地付款；
  • 简便的充值方法，只需用现金，借记卡/信用卡和银行转账即可充值；
  • 电子钱包的大小每月高达RM5,000，可轻松将钱转给另外的TNG账户，支持全国135,000个商户网点付款。
