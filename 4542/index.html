活动预告|走进马来西亚 携手跨境产业链供应链发展机遇论坛

点击蓝字

关注我们

自1974年中马正式建交以来，两国友好合作稳步发展。在政治、经济、文化和教育等多个领域，中马合作带来了许多机遇和益处。中马建交50周年标志着两国关系的深度和坚固，为未来的合作奠定了坚实基础。

为庆祝这一重要里程碑，助力中马两国供应链建设 ,促进中国企业高质量发展。马中关丹国际物流园、中国建设银行（马来西亚）分行、中国建设银行纳闽分行、中国建设银行宁波市分行联合主办的“走进马来西亚-跨境产业链供应链发展机遇论坛”将于3月26日在浙江宁波正式拉开帷幕。

本次论坛是中马两国供应链全产业链的开年盛会，也是一站式了解马来西亚当地营商环境、当地投资政策，寻找中国品牌出海和投资建厂的新机遇的理想之地。论坛以“跨境产业链供应链发展及出海”为主题，旨在深入探讨供应链融合发展实施路径，助力中国新能源、汽车制造、高端制造等行业供应链上中下游融合发展。

活动详情 Event Details

会议时间：2024年3月26日（13:30-17:30）

会议地点：中国建设银行(建行宁波市分行营业部)

会议主题：跨境产业链供应链发展机遇

图|马来西亚

活动亮点 Event Highlights

重量级嘉宾齐聚：马来西亚驻沪领事馆投资领事、戴德梁行跨境贸易研究专家及翰林律师事务所专业律师等团队，为来访嘉宾解读马来西亚招商投资政策，包括马来西亚重点产业、重点园区情况和优惠政策等。

行业闭门研讨分享：马来西亚政府单位代表、行业专家、企业代表以及多领域投资人齐聚，除了聚焦于中马贸易投资环境，深度剖析其中的机遇与挑战，更围绕智能制造、新能源产业、汽车零部件产业、石油和化工行业、科技创新等领域的最前沿话题展开深度探讨。另外，专家们将分享中国企业出海到马来西亚的丰富经验，涉及法律、金融服务、落地服务等综合层面，为参会嘉宾呈现高端服务格局，为中马合作架构了坚实稳定的桥梁。

图|马来西亚

论坛议程 Forum Agenda

13:30-14:00：参会嘉宾签到

14:50-15:10：产业链供应链全球化机遇分享

15:10-15:40：建设银行跨境金融服务分享

15:40-15:50：会间茶歇

15:50-16:10：马来西亚（法律与税务）摘要

16:10-16:50：圆桌讨论及问答环节

16:50-17:30：参会嘉宾与园区交流、互动

报名流程 Application

我们欢迎各行业企业以及意向合作伙伴参与本次论坛，请您发送“姓名+职务+联系方式”至邮箱：enquiries@mckilp.com。

报名审核通过后，我们将与您取得联系并向您发送参会确认函。

本次论坛为行业免费分享论坛，席位有限，欢迎有意向的伙伴积极报名。

作为本次论坛的主办单位，马中关丹国际物流园属于中马两国领导人直接倡议并推动落实的国家级产业园——马中关丹国际物流园三期项目，坐落于马来西亚半岛东海岸政治、经济、商业、交通和旅游中心——关丹市。园区紧邻东海岸最大港口关丹港、马来西亚东海岸铁路和关丹机场，是连接中国与东盟国家的国际贸易陆海新通道。园区还专门设立了项目落地协调服务团队，提供一站式便捷服务，帮助入园企业调研营商环境；协助落实园区优惠政策申请、推动政府行政审批执行；加快项目落地投产，提高生产经营效率。

谈合作、拓商机、建生态，让合作和出海更加高效便捷。期待3月26日与您“走进马来西亚-跨境产业链供应链发展机遇论坛”，与您相见，期待合作！
