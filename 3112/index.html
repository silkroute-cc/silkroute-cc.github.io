2040吉隆坡交通蓝图 不支持建新大道

（吉隆坡27日讯）《2040吉隆坡交通蓝图》不支持新大道计划！

吉隆坡市长拿督马赫迪今早在吉隆坡市政厅大厦推介《2040吉隆坡交通蓝图》时，这么指出。

他说，根据《2040吉隆坡交通蓝图》所点出的问题，隆市现阶段其实不支持再有新的大道进入，除非是现有大道之间的衔接。

他指出，无可否认，目前的一些大道出现“中断”的情况。

他举例，南北大道的南段与北段之间没有衔接，所以存在“迷失路段”。

可衔接现有大道

他说，换言之，若是涉及大道与大道之间的衔接兴建计划就可进行。

他说，若再有新的大道计划，会加剧车流量，导致更多私家车因方便而进入市区，届时只会导致道路服务达到很差的水平。

出席者有吉隆坡执行总监拿督阿兹米和基设规划小组主任诺法仄等。

50迷失公路9道路须提升

《2040吉隆坡交通蓝图》已鉴定隆市共有50条迷失公路，及9条现有道路必须提升！

马赫迪指出，只要上述道路获得提升及衔接，就可有效使用相关路段，尤其可发挥疏散交通的作用。

他指出，所有新的发展工程，必须以使用公交为优先，因公交可容纳大量的乘客，相比依赖使用私家车。

他说，隆市目前交通阻塞的情况必须改善，因会带来负面影响，包括二氧化碳排放量提高，以及导致外资却步等。

根据市政局提供的资料，涉及的迷失公路包括美罗柏兰岭西路（Jalan Metro Perdana Barat）衔接甲洞大路、增江北区北篱笆路（Jalan Benteng Utara）至第二中环公路（MRR2，往峇都方向）等。

至于9条需要提升的道路则包括都巴刹路（Jln Sentul Pasar）、5/51F路、达玛斯2路、泗岩沫达南路（从6/38J路交界处）至加拉4路（Jln Kiara 4，国际学校处）、杜达加拉路（Jln Duta Kiara）、哈兹韩沙路（Jln Hj Hamzah）、巴林峇燕路（Jln Palimbayan）、斯里本查拉路（Jln Sri Penchala）和士马勒阿比4路（Jln Semarak Api 4）。

2040年前不征入城费

2040年前都不会征收入城费！

根据早前报道，《2040吉隆坡交通蓝图》草案中，曾提及将落实入城费（Caj Kesesakan），不过由马赫迪周四推介的蓝图中，并没触及入城费一事。

马赫迪受询时指出，上述蓝图并未提及入城费。

他说，根上述蓝图的建议和规划，在2014年，一旦吉隆坡所有公共交通系统的建设和提升已完成及完善，就必须探讨届时的交通情况。

他说，如果依然面对塞车问题，就不排除需讨论征收入城费。

微型交通存数弱点

此外，询及微型交通（Micromobility）的使用情况，他则指出，这必须先采用实验计划的模式。

他指出，他了解其他市区会鼓励采用微型交通，不过微型交通存有数个弱点，尤其必须考虑安全因素，及不会影响其他路面的道路使用者。

“所以我们必须小心谨慎，尤其会触及交通法令，因此必须使用实验计划的模式进行。

私家车比例降25％

马赫迪指出，《2040吉隆坡交通蓝图》其中一项建议即是于2025年，将私家车的使用比例下降25%，同时不鼓励再有新的大道计划进入隆市，相信这绝对是个大挑战！

他说，吉隆坡目前的公共交通和私家车比例是25：75，完全可以显示隆市的交通非常阻塞，尤其是早上和下午高峰时段，道路服务水平达到E级，因路上有过多的私家车。

“我们必须采取缓解措施，否则隆市主要道路的道路服务水平是不排除会达F级，这是交通最糟糕的地步，为此市政厅推介《2040吉隆坡交通蓝图》，并提供许多建议和方案。

马赫迪指出，《2040吉隆坡交通蓝图》目标是在2025年，将公共交通和私家车比例达为50：50、2035年则是60：40、及2040年是70：30，希望隆市主要道路的服务水平为此可达C或D级。

他指出，为了达到上述目标，隆市政厅与许多单位配合，包括交通部、工程部、国家基建公司、马来西亚铁道公司、警方、陆路交通局和大马道路安全研究院等。

他指出，根据《2021年交通指数报告》，日本、马尼拉、耶加达和曼谷，分别排在全球17、18、46和74，反观马来西亚其实只排在210位。

他相信，一旦采取合适的应对方案，相信我国的塞车情况有待改善。

平衡使用公共系统 减少对私家车依赖

《2040吉隆坡交通蓝图》点出最重要即是平衡使用公共系统！

马赫迪指出，若要减少大众对私家车的依赖，最为关注的即是公交网络，尤其是铁轨、巴士、德士和电召车服务等，一律都必须提升至“第一英里和最后一英里”。

他说，该蓝图也建议在靠近各类公交系统，提供“泊车转乘”（Park N Ride）泊车场，方便民众采用公交入城。

“只要大众在高峰时段都是用公交，即可减少对私家车的依赖。”
