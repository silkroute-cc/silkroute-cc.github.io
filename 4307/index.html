打卡雪兰莪的四间咖啡馆

雪兰莪最近新增了不少不同风格的咖啡厅，如森林系、白色简约风、工业风、日系、复古风等，是不少年轻朋友爱打卡的地方。其中以超萌浣熊为主题的Bandit咖啡厅、外形像玻璃屋的Butty Boy Cooks 、90年代摩登怀旧风情的Guan's Kopitiam源茶室、工业风的Three Plates Full 人气颇高，来到雪兰莪旅行，记得要来这里拔草喔！

//

//

Bandit

位于PJ SS2的咖啡厅Bandit是一家以萌萌的小浣熊为主题的咖啡厅，结合原木质感元素点缀，主题空间充满英式复古的精致韵味。这里连门把都花了巧思做成了小面包的造型，连菜单，食物纸盒上都是浣熊的模样。这里也有售卖小浣熊的周边产品，如布包，T恤等，呆萌的模样随时融化你的。在饮食方面，餐厅提供各式西式餐点和咖啡，适合大家周末早餐到访！不妨尝尝他家主打的日本牛奶吐司，分别有巧克力、抹茶红豆、黑芝麻，三款口味选择。小编推荐你尝尝“Five My Way”，你可以自选5种配料烹制而成，份量足味道好，给你一个能量满满的周末！

//

//

Butty Boy Cooks

Butty Boy Cooks 的建筑外形看起来像一座玻璃屋，而店内设计以简约工业风设计为主题风格，再以绿植点缀，整体风格小清新。这家店的主打餐点的风格以泰式和西式融合，尤其以泰式鸡扒和冬荫鸡披萨最为出名！另外泰国道地的Mama’s Boy 海鲜汤面，再到经典风味的Pha Phed海鲜，这里一应俱全，喜爱泰式辣食的你别错过。他家的招牌甜点原味芝士蛋糕也值得学一尝，如果您是抹茶爱好者，Niko Neko抹茶一定会惊艳你的味蕾。

//

//

Guan's Kopitiam

于PJ Taman Rasa Sayang全新开张的Guan's Kopitiam源茶室，将道地传统的咖啡老店特色与现代咖啡厅融为一体，糅合了新旧元素，令人眼前一亮。他们家的装潢融合了90年代和摩登茶餐厅的怀旧元素，复古感满满，店内随便一个角落都是拍写真的好地方。在饮食方面，这家咖啡厅主打经典的道地早餐和烘培甜品，如咖喱鸡椰浆饭、东南亚传统料理暹罗米粉与咸蛋可颂深受食客欢迎。

//

//

Three Plates Full

新加入SS15社区的 Three Plates Full ，餐厅外观朴实无华，但踏入餐厅后你会发现这是一个舒适的用餐空间。Three Plates Full 共有三层楼，它以工业风为主，采用了大量藤制椅子，木质和大理石桌子装饰点缀，氛围温馨典雅，空间清新明亮，各个角落都很适合拍照。这里提供早午餐，午餐和晚餐，主打各式西餐，必点的菜式有 Devesa Devesa 牛里脊肉、炸薯球、脆皮猪排、烤鲷鱼。
