浮罗山背--前所未有的新体验

在浮罗山背可以与大自然亲密接触，

一路上踏着脚车，

穿过绿油油的稻田，经过红树林，螃蟹洞！

浮罗山背 Kampung Terang 稻田的货柜艺术装置品，获“街头艺术城市”（Street Art Cities）国际平台评选为2021年1月全球最佳街头艺术品。这片稻田随时欢迎您的到来，感受没有钢筋水泥建筑遮挡，在蓝天白云衬托下打卡稻田货柜！

货柜装置分为前后两面，分别由国际艺术家 Marat Danivan和本地艺术家Leonard Siaw 及 Jagung 所创作。本地艺术家所创作的 《The Jov Of Harvest》 呈现的是，尽管现代和先进的机械已逐渐取代了许多人工耕种任务，但浮罗山背的稻农仍采用传统的手工种植和收割水稻的手法工作，只希望确保每个季节都能提供优质的收成。作者通过作品向所有勤劳的稻农致敬。

浮罗山背壁画 Street Art

来槟城就该寻找经典的壁画，只要一路沿着镇里的街道就会看见一幅幅的壁画在古老的建筑物上。虽然这里的壁画不像乔治市（George Town）壁画分布在大街小巷内，这里拥有的5幅壁画都集中在大街上，分别是出自立陶宛艺术家尔纳斯（Ernest Zacharevic）的《打金匠》，以及另外来自俄罗斯斯伯利亚的女画家Julia Volchkova之手的《补渔网老人》、《马来武术》、《客家女人》、《割橡胶的女人》，非常方便游客寻找及拍照留念。

直落巴巷景点多-主题乐园王国

槟城热带果园

//

直落巴巷有许多有趣好玩的旅游景点，其中，槟城热带果园主打自然旅游，是适合一家大小体验热带水果的最佳地点。种植的热带与亚热带地区果树超过250种，具丰富的资源和自然美景，是大家不能错过的景点。随着不同的水果季节，能在不同的时候，欣赏到不同的果树，了解水果的各种知识，同时品尝不同的热带水果。可以在直落巴巷逗留数日，体验不同的槟城特色！

Escape主题乐园

//

全世界最长的滑水道，就在槟城直落巴巷“世外逃园”Escape主题乐园。被评为2018年马来西亚旅游大奖 (Malaysia Tourism Awards）最佳旅游景点之一。这项全长1111公尺，需要大约4分钟才能滑完全程的滑水道，2019健力士世界纪录评审员彼得的见证下，获颁世界最长滑水道的证书，扬威国际。一个能满足所有人“内心小孩的渴望”的游乐设施，亲近大自然，并且爱上大自然。

虫鸣大地 ENTOPIA

//

槟城Entopia 虫鸣大地蝴蝶公园坐落在一个巨大的玻璃温室内、可以看到蝴蝶满天飞舞的地方。这里是热带地区上第一个蝴蝶和昆虫保护区，15,000多只自由飞翔的蝴蝶、 种类繁多的昆虫和爬行动物以及200 多种植物群的安全家园。

热带植物香料园

//

热带植物香料园隐藏在直落巴巷临海的依个静谧山谷里，总占地八英亩，已开发面积 4.5 英亩，其余的都保持原本的生态面貌香料园在2003年造园时，在保存原内原生植物的原则下制定规划，使这个条里有序的园林仍保留着原始幽深的气息，成为特别是踏进园林里，扑鼻而来的各种香草，植物的味道，马上就让人神经大为松弛。
